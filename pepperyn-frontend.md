# Pepperyn — Frontend MVP (Next.js + Tailwind)

## 🎯 OBJECTIF

Construire une interface simple, premium et orientée décision permettant :
- upload de données financières
- affichage d’une analyse claire
- perception de valeur immédiate

---

## 🧱 STACK

- Next.js (App Router)
- Tailwind CSS
- TypeScript
- API calls vers backend (Vercel / Supabase)

---

## 📄 PAGES

### 1. Landing Page (/)

#### Hero Section

Titre :
Scanner financier pour entreprises

Sous-titre :
Transformez vos données financières en décisions business

Texte :
En quelques secondes, identifiez ce qui fonctionne, ce qui vous coûte et ce que vous devez faire ensuite - revenus, coûts, marges, anomalies et recommandations activables. 

Bullets :
- ce qui vous fait gagner de l’argent
- ce qui vous en fait perdre

- mémoire persistante

Micro preuves :
✔ Analyse structurée
✔ Recommandations concrètes
